# Movie Analytics Pipeline (AWS + Snowflake + Power BI)

## Overview
Built an end-to-end data engineering pipeline using AWS S3 and Snowflake, and visualized insights using Power BI.

## Tech Stack
- AWS S3
- Snowflake
- SQL
- Power BI

## Features
- Data ingestion from S3
- ETL pipeline in Snowflake
- Data cleaning and transformation
- Aggregated analytics table
- Interactive dashboard

## How to Run
1. Upload CSV to S3
2. Run SQL pipeline in Snowflake
3. Connect Power BI to Snowflake
4. Build dashboard using movie_analysis table
